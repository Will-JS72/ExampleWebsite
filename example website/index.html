About

  • responsive navbar.

    • bootstrap allows a navbar that resizes based on screen size, to try this out press ctr+shift+I or right click inspect and try resizing the active window.

  • navbar page scrolling.

    • try clicking the navbar links to scroll content into view. You can click the example logo text to return to the home area.

  • mobile friendly relative sizing and display.

    • bootstrap grid allows a content to resize based on screen size, to try this out press ctr+shift+I or right click inspect and try resizing the active window or open in your mobile browser.

  • input forms.

    • forms are validated using bootstrap4 and are sent to my email.
